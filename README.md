# Tech Blog

  [![License](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
  
  ## Description
  
  My motivation to build this project is to build a CMS-style blog site. Now this project will publish articles, blog posts, and my thoughts and opinions. This project solves the issue of having one consolidated location for all of my writings about tech.

  ## Installation
  
  Prerequisites: Node.js
  
  Packages: express-handlebars, MySQL2, Sequelize, Express.js, bcrypt, connect-sessions-sequelize,dontenv, express-session
  
  ## Usage
  
  Make sure you have all the prerequisites and packages installed. 
  
  - Click on the link of the deployed application
    
    [Tech Blog](https://ancient-ridge-55525.herokuapp.com/)
